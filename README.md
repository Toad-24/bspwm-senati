```bash
git clone https://github.com/xJackSx/BSPWMparrot.git

```

## Entorno BSPWM orientado a estudiantes de SENATI

Esta personalización de BSPWM está diseñada para brindar un entorno ligero, rápido y productivo para estudiantes de SENATI, manteniendo la filosofía minimalista de Linux y añadiendo accesos rápidos a herramientas académicas.

### Características

* Entorno basado en BSPWM y Polybar.
* Tema inspirado en los colores institucionales de SENATI.
* Accesos directos al Campus Virtual de SENATI.
* Rofi personalizado para aplicaciones y utilidades académicas.
* Optimizado para máquinas virtuales VMware y VirtualBox.
* Consumo reducido de recursos.
* Configuración orientada a programación, redes, ciberseguridad y automatización.
